# Validador de Bandeira de Cartão de Crédito




## 💼 Sobre o projeto

Este repositório foi criado para o projeto **"Criando um Validador de Bandeiras de Cartão de Crédito com o GitHub Copilot"** do bootcamp da **DIO**.  
O objetivo é aplicar os conceitos aprendidos e explorar como a inteligência artificial pode acelerar o desenvolvimento e aumentar a produtividade.

---

## ✅ Objetivo

Validador de Bandeira de Cartão de Crédito utilizando **JavaScript** e **GitHub Copilot**.
