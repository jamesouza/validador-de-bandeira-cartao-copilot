# Identificador de Bandeira de Cartão de Crédito

## 📖 Descrição
Este projeto tem como objetivo desenvolver uma aplicação simples capaz de identificar a bandeira de um cartão de crédito (Visa, MasterCard, American Express, Discover, etc.) com base no número informado.  
O desafio foi proposto pela **DIO (Digital Innovation One)** e recriado/aprimorado com o auxílio do **GitHub Copilot**, explorando como a inteligência artificial pode acelerar o desenvolvimento e aumentar a produtividade.

## 🛠️ Funcionalidades
- Identificação da bandeira do cartão de crédito a partir do número.
- Suporte às principais bandeiras:
  - **Visa** → começa com 4  
  - **MasterCard** → começa com 51–55 ou 2221–2720  
  - **American Express** → começa com 34 ou 37  
  - **Discover** → começa com 6011, 622126–622925, 644–649 ou 65  
- Retorno de mensagem caso a bandeira não seja identificada.
